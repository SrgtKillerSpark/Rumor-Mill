# Marketing Screenshots

Drop captured game screenshots here.

## Naming convention

```
asset-[letter]-[slug]-[YYYY-MM-DD].png
```

Examples:
- `asset-e-act-state-npc-2026-04-30.png`
- `asset-f-inquisitor-pressure-s4-2026-04-30.png`
- `asset-g-post-scenario-analytics-s1-2026-04-30.png`
- `asset-h-defending-cascade-2026-04-30.png`
- `asset-i-noble-estate-night-2026-04-30.png`
- `asset-j-s3-dual-track-split-2026-04-30.png`

## Technical requirements

- **Resolution:** 1920×1080 minimum
- **Format:** PNG for screenshots
- **Build:** Must be from post-SPA-410 art pass build (`builds/RumorMill.exe`)
